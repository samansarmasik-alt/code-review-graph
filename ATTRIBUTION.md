# Attribution and provenance

ForceGraph is based on the open-source
[`tirth8205/code-review-graph`](https://github.com/tirth8205/code-review-graph)
project and continues to use substantial portions of its implementation,
documentation, tests, and repository history.

The upstream project is distributed under the MIT License. Its original license
and copyright notice are preserved in [`LICENSE`](LICENSE). ForceGraph is also
distributed under the MIT License.

## Upstream relationship

- Upstream repository: `https://github.com/tirth8205/code-review-graph`
- Initial ForceGraph baseline: upstream release `v2.3.7`
- Initial baseline commit: `6a1ee1c`
- ForceGraph repository: `https://github.com/samansarmasik-alt/code-review-graph`

ForceGraph is an independent community development fork. It is not presented as
an official release of, or endorsement by, the upstream maintainer.

The upstream README at the time of the fork is retained in
[`docs/UPSTREAM_README.md`](docs/UPSTREAM_README.md) for feature documentation and
historical reference.

## Contribution policy

When upstream changes are incorporated, their authorship and commit history
should be preserved whenever practical. New ForceGraph features should be clearly
identified in release notes so users can distinguish them from upstream
capabilities.
